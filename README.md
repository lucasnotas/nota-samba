<h1>Ambiente virtual: </h1>
<p>Entar na pasta do Ambiente e execute:</p>

  	python3 -m venv venv
<br/>
<p>Ativar Ambiente Virtual(Loga pelo Terminal)</p>

	source venv/bin/activate
<br/>
<p>Desativar Abiente Virtual</p>

	sudo systemctl restart smbd
<br/>
<p>Utilizando pip dentro do ambiente virtual:</p>

 	./venv/bin/pip install nome_biblioteca
</br>
